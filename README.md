# Bay Ridge Bicycles Mobile Preview

Static, mobile-first preview concept for emailing/texting to Bay Ridge Bicycles.

- Not the official Bay Ridge Bicycles website.
- Demo form is intentionally non-submitting; it shows the live request-flow concept.
- `robots.txt` and meta robots are set to noindex/nofollow/disallow.
- Built outside the old Sigma methodology because this is a same-day revenue preview.
